# Hackster.io Project Template

## Basics

- **Project Name** : Inclusive Maker - Brain-Controlled Hand Glove
- **Pitch** : A low-cost brain-computer interface glove controller to help people with hand paralysis open and close their hands.
- **Image** : Photo du prototype / schéma conceptuel

## Things

### Hardware

- Unicorn Hybrid Black EEG headset
- Arduino / Raspberry Pi
- Servomoteur(s) ou gants motorisés
- Câbles et breadboard

### Software

- Python 3.10+
- g.Pype SDK
- pylsl
- NumPy / SciPy / scikit-learn
- PySide6 (interface graphique)

### Tools

- VS Code
- Git / GitHub
- Soldering iron
- 3D printer (optional)
