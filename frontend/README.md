# Frontend - Inclousive Maker

Interface utilisateur pour le démonstrateur BCI.

## Idées

- Dashboard temps réel affichant l'état détecté et la dernière commande envoyée.
- Affichage du signal EEG et des bandes de puissance.
- Boutons manuels pour simuler OPEN / CLOSE / IDLE.

## Stack proposé

- HTML + JavaScript + WebSocket (léger, cross-plateforme)
- Ou PyQtGraph via g.Pype si intégration directe.
